# Projet de cycle de vie (équipe 18)
Projet d'application de conversion entre des chiffres romains et chiffres arabes

## Stack
- NodeJS (express)
- Redis

## Pré-requis
- Récupérer les variables d'environements auprès d'un membre de l'équipe
- Téléchargez et installez [Git](https://git-scm.com), [Redis](https://redis.io/) et [NodeJS](https://nodejs.org)


## Lancement en local
1. Cloner le projet:
```shell
git clone git@github.com:ilies-t/cycle-vie-roman-arabic-roman.git
```

2. Accéder au répertoire du projet:
```shell
cd cycle-vie-roman-arabic-roman
```

3. Installer les dépendances :
```shell
npm install
```

4. Démarrer le projet :
```shell
npm run run:dev
```
